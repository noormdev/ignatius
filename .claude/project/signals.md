# Project signals

## Framework & runtime

- Runtime: Bun (all scripts, server, test runner, binary compiler)
- Language: TypeScript (strict, ESM modules, `type: "module"` in package.json)
- Frontend: React 19, Cytoscape.js 3.31 + cytoscape-elk 2.2 + cytoscape-navigator 2.0, ELK layout engine
- Markdown parsing: markdown-it 14; YAML parsing: yaml 2.8
- No Express, no Vite, no webpack ‚Äî Bun.serve + Bun HTML imports only
- Dev tools: Playwright (screenshot/SSE tests), webview-bun

## Domain detail

### cli

[`src/cli/cli.ts`](../../src/cli/cli.ts) is the binary entry point. Eight subcommands registered: `serve`, `server` (alias for serve), `dict`, `graph`, `validate`, `flow`, `export`, `version`, `update`. `dict`, `graph`, and `flow` are removal stubs ‚Äî each prints "Removed ‚Äî use: ignatius export -o model.html" to stderr and exits 1. Active model subcommands: `serve`, `validate`, `export`.

`serve` accepts optional positional `[path]` and `--port`/`-p`, `--model`, `--open`/`-o` flags. Delegates binding to `serveWithPortFallback` in [`src/cli/serve-port.ts`](../../src/cli/serve-port.ts). When the requested port is taken (`EADDRINUSE`), TTY prompts via `@clack/prompts` text defaulting to the next free port; non-TTY auto-advances. After binding, `--open` dynamically imports [`src/cli/open-browser.ts`](../../src/cli/open-browser.ts) and calls `openBrowser`.

`export` subcommand: parses entity model via `parseModels()`, validates with `validateModel()`, then parses flows via `parseFlows()` if a `flows/` directory exists, validates with `validateFlows()`. Loads the embedded React bundle via `loadEmbeddedBundle()`, calls `generateApp(model, flowModel, bundle, { themeMode })` from [`src/generators/app.ts`](../../src/generators/app.ts), writes a single HTML file. Exit code 1 when any entity global errors OR flow Class-B errors are present.

`validate` subcommand: same `parseModels` ‚Üí `validateModel` ‚Üí optional `parseFlows` + `validateFlows` flow, prints findings to stderr, prints a one-line stdout summary (`‚úì`/`‚úó`). Exit code 1 on errors.

[`src/cli/open-browser.ts`](../../src/cli/open-browser.ts) exports `browserOpenCommand(platform, url): string[]` (pure) and `openBrowser(url, platform?)` (fire-and-forget `Bun.spawn`). Dynamically imported only on `--open`.

[`src/cli/version.ts`](../../src/cli/version.ts) exports `VERSION` from a JSON import of [`package.json`](../../package.json) ‚Äî Bun inlines it at `--compile` time.

[`src/cli/update.ts`](../../src/cli/update.ts) powers `update` (flags `--check`, `--yes`/`-y`). Resolves latest tag via `releases/latest` redirect Location header, compares semver, on consent downloads + verifies sha256, atomically renames over `process.execPath`. Guards: dev runtime ‚Üí no self-replace; win32 ‚Üí manual-download message; non-TTY without `--yes` ‚Üí report-only.

[`src/cli/discover.ts`](../../src/cli/discover.ts) ‚Äî pure model-root resolver. Exports `resolveModel(base, opts): Promise<ResolveResult>` and `ModelCandidate` / `ResolveResult` types. Algorithm: (1) base itself has `ignatius.yml` ‚Üí single; (2) search down (skipping `_*`, `node_modules`, `.git`, `dist`, `tmp`, [`trash`](../../trash), `.worktrees`, [`.claude`](..)); (3) walk up; (4) exactly 1 ‚Üí single; (5) multiple + `--model` ‚Üí filter; (6) multiple + no key ‚Üí many. `ResolveResult` discriminated union: `single | many | no-match | none`.

[`src/cli/resolve-model.ts`](../../src/cli/resolve-model.ts) ‚Äî exports `pickModel(base, modelKey): Promise<string>`. Handles all four result kinds: single ‚Üí return dir; none/no-match ‚Üí stderr + exit 1; many + non-TTY ‚Üí stderr key list + exit 2; many + TTY ‚Üí `@clack/prompts` select picker (cancel ‚Üí exit 130).

### server

[`src/server/server.ts`](../../src/server/server.ts) exports `serveCommand(modelsDir, opts)` returning `{ server, stop }`. Routes:

- `GET /` ‚Üí bundled React HTML (unified SPA)
- `GET /dict` ‚Üí 302 redirect to `/#view=dict`
- `GET /flow` ‚Üí 302 redirect to `/#view=flow`
- `GET /flow-dict` ‚Üí 302 redirect to `/#view=dict` (CP5 ‚Äî process dictionary fused into Dictionary view)
- `GET /api/model` ‚Üí JSON `{ model, parseGlobalErrors, validation, layoutKey }` where `validation = { entityErrors, globalErrors, cleanedModel }` and `layoutKey = layoutFingerprint(model)`
- `GET /api/flow` ‚Üí JSON `{ diagrams, entityModel, validation, flowLayoutKeys }`. Guard: if no `flows/` directory returns empty-state 200. `entityModel` travels with the payload so the flow viewer can resolve `db:` store docs to ERD entity narratives.
- `GET /api/asset` ‚Üí model-dir asset proxy (path traversal blocked)
- `GET /events` ‚Üí SSE stream; timeout disabled via `server.timeout(req, 0)`

`fs.watch` watches `modelsDir` recursively; only `.md` and `.yaml` extensions trigger events. Debounce: 200ms coalesce. SSE event name: `model-changed`. `stop()` closes the watcher, clears debounce timer, clears SSE client set, stops Bun server.

### parser

[`src/model/parse.ts`](../../src/model/parse.ts) exports `parseModels(dir): Promise<ParseResult>`. `ParseResult = { model: Model; globalErrors: GlobalError[] }`. Config loading: reads `ignatius.yml`; top-level keys populate `_meta`; `theme:` deep-merged via `mergeTheme()`; `branding:` merged via `mergeBranding()`. `Model` type: `{ groups, nodes, edges, subtypeClusters, theme, branding, _meta? }`. Entity classification fully derived (5-rule order): Classifier ‚Üí Subtype ‚Üí Associative ‚Üí Dependent ‚Üí Independent. `identifying` per edge also derived. `deriveCardinality()` uses derived `identifying` + nullability + AK membership. Body markdown rendered to HTML via markdown-it at parse time.

**Folder model (v0.11.0, #16):** Entity discovery scans `data/**/*.md` exclusively ‚Äî the old `**/*.md` + `_`-segment-skip + `flows/`-skip logic is gone. `groups/` is optional ‚Äî absent means zero groups, no throw. Files outside `data/` (at the root, in `notes/`, etc.) are never parsed as entities. The five recognized top-level folders are `data/`, `flows/`, `groups/`, `externals/`, and `stores/`; everything else is free-form and ignored.

**Wiki-links (two-pass body rendering):** `ModelNode` carries `bodyLinks?: string[]`. Body rendering deferred to a second pass after all entity ids are known. [`src/model/wikilink.ts`](../../src/model/wikilink.ts) exports `WikiLinkEnv`, `splitWikiTarget`, and `wikiLinkPlugin(md)`. Valid links emit `<a class="entity-link" data-entity="‚Ä¶">‚Ä¶</a>`; unknown links emit `<span class="entity-link entity-link--missing" ‚Ä¶>`. Absent `knownIds` ‚Üí optimistic.

**Predicates:** `Predicate = { fwd: string; rev: string }` and `normalizePredicate(raw): Predicate`. `ModelEdge.predicate` is always a normalized `Predicate`.

**ModelIndex ([`src/model/model-index.ts`](../../src/model/model-index.ts), 222L):** exports `buildModelIndex(model: Model): ModelIndex` and `endpointKey(source, target): string`. Pure module ‚Äî no Bun/Node/DOM imports; browser-safe; same discipline as `validate.ts`. `ModelIndex` carries 13 maps:

- `nodeById: Map<string, ModelNode>` ‚Äî O(1) node lookup by id
- `nodeIdSet: Set<string>` ‚Äî O(1) membership tests
- `edgesBySource: Map<string, ModelEdge[]>` ‚Äî edges keyed by source node id
- `edgesByTarget: Map<string, ModelEdge[]>` ‚Äî edges keyed by target node id
- `edgeByEndpointPair: Map<string, ModelEdge>` ‚Äî edge keyed by `"source>target"` token (produced by `endpointKey`)
- `pkByNode: Map<string, string[]>` ‚Äî pk column names per node
- `columnsByNode: Map<string, Record<string, ColumnDef>>` ‚Äî declared columns per node
- `akColumnsByNode: Map<string, Set<string>>` ‚Äî AK-participating column names per node (absent when no AKs)
- `fkColumnsByNode: Map<string, Set<string>>` ‚Äî FK column names (from `edge.on` keys) per source node (absent when no outgoing edges)
- `subtypeMemberToCluster: Map<string, SubtypeCluster>` ‚Äî first-wins cluster for a member id
- `clustersByMemberId: Map<string, SubtypeCluster[]>` ‚Äî all clusters for a member (multi-cluster support)
